##Comando usado
 git worktree add -b feature-b ../wt-feature-b main

##La salida
/home/artur/Documentos/actividad-6-13-git-worktree  f7c6dba [main]

##EXPLICACIÓN
git branch solo crea la rama.
git worktree add -b crea la rama y además te da otra carpeta para trabajar en ella.
